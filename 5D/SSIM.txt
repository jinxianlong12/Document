설치--------------------------->
1. 폴더 복사
2. 라이센스 신청
3. mysql 설치, 비번 topcore1357
4. 스크립트

OHT--------------------------->
1. 새 프로젝트를 만든다. file-->new-->프로젝트네임, 너비, 길이 등을 입력하고 Building Import버튼을 클릭해서 바닥을 선택한다.
2. Map Drawing을 클릭하고 OHT노선을 설계한다.
3. Setting을 클릭하고 EQ를 선택한다음 Location을 클릭하여 EQ를 배치한다.
4. select Mode에서 Road Point Mode를 선택하고 OHT위치를 클릭한다음 오른쪽마우스 클릭 하고 EQ Road Connection Set을클릭하여 EQ랑 매핑한다.
5. OHT를 추가한다.
6. select Mode에서 Road Point Mode를 선택하고 오른쪽 마우스 클릭하여 muiti Mode에서 Muiti Command Mode를 선택하고 OHT이동 및 Lot Pick Place로직을 지정한다.
7. 경로 지정완료후 Object Mode로 변환하고 OHT를 클릭하고 오른쪽마우스로 Set Return Route를 클릭하고 팝업창에서 속도를 입력하고 Apply를 클릭후 ok버튼을 클릭한다.
8. Lot은 생성후 설비 안의 위치에 드래그 하여 놓는다.
9. 3D.exe를 실행하여 결과를 확인한다.

AGV--------------------------->
1. AGV Road Drawing을 선택하여 AGV로선을 설계한다.
2. 화면에 설비, Lot, AGV등을 추가 한다.
3. OHT모드처럼 이동경로를 지정하고 command명령어를 등록한다. OHT와 같음

반송라인--------------------------->
1. 설비를 일정한 간격에 의해 배치하고 이동할 Lot을 설비 안에 드래그 한다.
2. 컨베이어를 배치한다. 3D뷰어를 켜고 설계한다.
3. Object Mode를 선택하고 설비부터 Lot이 이동할 컨베이어를 차례대로 클릭해준다.
4. 선택 다 하고 마우스 우클릭 Set Producing Route를 선택해준다.
